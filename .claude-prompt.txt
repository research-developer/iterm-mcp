## Issue #65: Add Hierarchical Task Delegation with Manager Agents

## Summary
Implement hierarchical task delegation where manager agents coordinate planning and validation, delegating specific tasks to specialist agents.

## Background
Hierarchical orchestration is a proven pattern:
- **CrewAI**: Manager agent coordinates planning, validates results before proceeding
- **AutoGen**: GroupChatManager orchestrates multi-agent conversations
- **Agency Swarm**: CEO > Developer > VA hierarchy with explicit flows

## Proposed Implementation

### Manager Agent
```python
from typing import List, Optional

class ManagerAgent:
    """Agent that coordinates other agents."""
    
    def __init__(
        self,
        name: str,
        workers: List[str] = [],
        delegation_strategy: str = "round_robin"
    ):
        self.name = name
        self.workers = workers
        self.strategy = delegation_strategy
    
    async def delegate(
        self,
        task: str,
        required_role: Optional[SessionRole] = None,
        validation: Optional[Callable] = None
    ) -> TaskResult:
        """Delegate task to appropriate worker."""
        # Select worker based on strategy and role
        worker = await self.select_worker(required_role)
        
        # Execute task
        result = await self.execute_on_worker(worker, task)
        
        # Validate if provided
        if validation and not await validation(result):
            raise ValidationError(f"Task result failed validation")
        
        return result
    
    async def orchestrate(
        self,
        plan: List[TaskStep]
    ) -> List[TaskResult]:
        """Execute a multi-step plan."""
        results = []
        for step in plan:
            result = await self.delegate(
                task=step.task,
                required_role=step.role
            )
            results.append(result)
            
            # Stop on failure unless step is optional
            if not result.success and not step.optional:
                break
        
        return results
```

### Task Definition
```python
class TaskStep(BaseModel):
    task: str
    role: Optional[SessionRole] = None
    optional: bool = False
    depends_on: List[str] = []
    validation: Optional[str] = None

class TaskPlan(BaseModel):
    name: str
    steps: List[TaskStep]
    parallel_groups: List[List[str]] = []  # Steps that can run in parallel
```

### Usage Example
```python
# Define manager with workers
orchestrator = ManagerAgent(
    name="build-orchestrator",
    workers=["build-agent", "test-agent", "deploy-agent"]
)

# Define plan
plan = TaskPlan(
    name="build-and-deploy",
    steps=[
        TaskStep(task="npm install", role=SessionRole.BUILDER),
        TaskStep(task="npm run build", role=SessionRole.BUILDER),
        TaskStep(task="npm test", role=SessionRole.TESTER),
        TaskStep(task="docker push", role=SessionRole.DEVOPS),
    ]
)

# Execute
results = await orchestrator.orchestrate(plan.steps)
```
